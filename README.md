# TP-Objetos
En este práctico aplicará programación orientada a objetos (POO) programando sus
propias clases en C++ en la plataforma Arduino.
● Puede trabajar en un simulador de Arduino como Wokwi o Tinkercad o con el mismo
Arduino que ya posea.
● Recuerde comentar su código fuente.
● En la entrega adjunte el diagrama o esquema de conexión de componentes y el código
fuente.
